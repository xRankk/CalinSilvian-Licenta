# CalinSilvian-Licenta
Repository public pentru proiectul de licență, în care apar fișierele sursă folosite pentru sistemul de control si actiune. Codul este făcut pentru flatforma ESP32 Dev Module și în IDE-ul Arduino IDE.
Acest repository conține atât codul pentru microcontrolerul secundar (ESP32) cât si pentru microcontrolerul de pe placa principală (ESP32). Toate schemele hardware se regăsesc în documentația licenței și pe EasyEda (link-uri : https://easyeda.com/editor#project_id=6913172ab09c4deb8135c8c763efd244 (punte H) și https://easyeda.com/editor#project_id=b3013229778641448f07012e25e5aba7 (placa centrală)).
Codurile au fost scrise cu următoarele reguli: cammelCasing și brief-uri pentru fiecare funcție declarată și folosită.
